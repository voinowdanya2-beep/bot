# Manicure Telegram Bot

## Запуск

1. Открой папку проекта в PyCharm.
2. Создай виртуальное окружение:
   ```bash
   python -m venv venv
   ```
3. Активируй окружение.

Windows:
```bash
venv\Scripts\activate
```

macOS/Linux:
```bash
source venv/bin/activate
```

4. Установи зависимости:
```bash
pip install -r requirements.txt
```

5. Переименуй `.env.example` в `.env` и вставь свои данные.

6. Запусти:
```bash
python bot.py
```

## Команды

/start — меню клиента  
/admin — админ-панель
